Insertion Sort

1.Soru:

Verilen dizi: [22, 27, 16, 2, 18, 6]
Insertion Sort aşamaları:
[22,27,16,2,18,6]
[22,27,16,2,18,6]
[16,22,27,2,18,6]
[2,16,22,27,18,6]
[2,16,18,22,27,6]
[2,6,16,18,22,27]

Big-O gösterimi:

1 + 2 + ... + (n-1) + n

= (n * (n+1)) / 2

= (n² + n) / 2

= O(n²)


Avarge case kapmasına girer.
